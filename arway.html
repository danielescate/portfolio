ARWAY

Overview

ARWAY is a concept app that stands for Augmented Reality Wayfinder.

This project started out as ARWIN - Augmented Reality Window - see concept video. Because of limitations, it was decided to update the concept to a mobile wayfinder. ARWAY is meant to allow users to map out a route and view directions in augmented reality. When they reach their destination, they can use ARWAY to view points of interest in AR. There is also a "Favourites" tab where users can store frequently visited locations.

The goal for this project was to continuously work on building prototypes throughout the semester. This prototype was created using React, Google Map Directions API, CSS and deployed to Expo Go.
